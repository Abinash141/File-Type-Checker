# File-Type-Checker
A file extension  checker that checks and verify the raw bytes and compare them to known signature of file types, It also shows 16 bytes HEX, also verify file type of ".jpeg" and ".png""

You can either build the "filecheck.go" by "go build .\filecheck.go" this will create  "filecheck.exe" an executable file.

Or you can run the following in command-promt  " go run filecheck.go "
